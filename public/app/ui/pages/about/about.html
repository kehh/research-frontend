About this Research Facility

This website is an SPA, which means that the user interface, including all pages and components, is generated and managed within your web browser. The application is written entirely in JavaScript, HTML and CSS, which are the underlying technologies for the entire www. See Wikipedia for more information about the SPA application architecture.

The data displayed by the application comes via web services provided by CA, an open source collections management application. See the CollectiveAccess website for more information.
